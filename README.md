# Superstore Sales & Profit Analytics Dashboard

## Project Overview

Analyzed 9,994 retail transactions using SQL and Power BI to evaluate sales performance, profitability, and regional trends.

## Tools Used

- SQL (MySQL)
- Power BI
- Excel

## Key Features

- KPI Tracking (Sales, Profit, Quantity, Profit Margin)
- Regional Performance Analysis
- Category & Sub-Category Analysis
- State-wise Sales Visualization
- Business Insights & Recommendations

## Key Insights

- West region generated the highest sales and profit.
- Technology category delivered the strongest performance.
- Higher discounts negatively impacted profitability.
- Consumer segment contributed the largest share of sales.
